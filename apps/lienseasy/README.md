# LiensEasy (product shell)

The thin product **LiensEasy** = Tower + the `lien-collections` module + the brand. A shell, not
a place for business logic: it wires the Tower platform to the activated module set and applies
the brand. Buying a second product later just activates more modules on the same Tower tenant —
no second app, no second database.

**Lands here:** the app entry/shell from LienGuard's `artifacts/lien-collections`
(`src/components/nav/AppShell.tsx` + bootstrap), reduced to a shell once feature screens move
into the module. Placeholder until P4.
